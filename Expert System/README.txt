How to load the expert system:

1. Unzip folder: "Expert System".
2. Enter directory: "pl" -> "bin".
3. Run file "plwin.exe". This is the Prolog development environment.
4. On top, click "file" -> "Consult".
5. Choose file "Expert.pl".
6. Enter command "load.".
7. Enjoy.
